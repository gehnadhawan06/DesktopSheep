# 🐑 Desktop Sheep

A tiny animated sheep that lives on your desktop. She wanders around, naps, gets excited when you click her, and loves being petted. Built with Python and PySide6.

## ✨ Features

- 💤 **Idle** — the sheep stands around blinking, just vibing
- 🚶 **Walk** — she autonomously wanders around your screen in random directions
- 🎉 **Bounce** — click her and she gets excited and bounces before going for a walk
- 😴 **Sleep** — if left alone long enough, she'll doze off on her own
- 🌞 **Wake** — click her while sleeping to wake her up, or she'll wake up on her own after a while
- ❤️ **Petting** — click and drag across her to pet her, hearts appear while you do
- ❄️ **Freeze Mode** — double-click to freeze her in place (great for meetings or screen sharing), drag her anywhere while frozen, then double-click again to unfreeze
- 🐑 **Baa Sound** — she baas when she bounces
- 📌 **Always on Top** — stays above all other windows
- 🖥️ **Never Leaves the Screen** — respects screen boundaries at all times

## 🚀 How to Run (Python)

### 1. Clone the repository

```bash
git clone https://github.com/gehnadhawan06/DesktopSheep.git
cd DesktopSheep
```

### 2. Create a virtual environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run

```bash
python main.py
```
